При востановлении Windows с помощью образа востановления хранящийся на удаленном сервере

1. С установочной флешки зайти в меню востановления. 
2. Выбрать пункт с удаленого компьютера. Дать поиск сети (что бы влючилась сетевая карта)
3. Зайти в консоль Shift + F10
4. ipconfig /all
5. netsh interface ip set address name="@@@Ethernet@@@et0" static IP MASK GATEWAY
6. Прописать путь \\server_backup\share
7. Логин server_backup\user
8. Пароль *******
9. Наслаждаемся тем, что все получилось(но эт не точно)
